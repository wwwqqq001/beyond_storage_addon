# Beyond Storage Addon

这是一个为 **Beyond Dimensions** 模组开发的扩展插件。

## 功能介绍
本模组为游戏增加了一个便捷功能：**一键存入维度网络**。
- 当玩家打开任何容器（如箱子、熔炉等）时，可以通过特定的网络包触发逻辑。
- 自动将容器内的物品转移到玩家所属的“维度网络（Dimension Network）”统一存储中。
- 提供屏幕提示，告知存入的物品组数或网络是否已满。

## 安装要求
- **Minecraft:** 1.20.1
- **Forge:** 47.2.0 或更高版本
- **依赖模组:** [Beyond Dimensions](https://www.curseforge.com/minecraft/mc-mods/beyond-dimensions)

## 开发说明
该项目使用了 Forge Gradle 6.0 进行构建，依赖了 Beyond Dimensions 的 API 进行维度网络的交互。

# Beyond Storage Addon (English)

This is an extension addon for the **Beyond Dimensions** mod.

## Features
This mod adds a convenient feature: **Quick Store to Dimension Network**.
- When a player opens any container (e.g., chests, furnaces), it can trigger a logic via custom network packets.
- Automatically transfers items from the container to the player's "Dimension Network" unified storage.
- Provides on-screen messages indicating the number of item stacks stored or if the network is full.

## Requirements
- **Minecraft:** 1.20.1
- **Forge:** 47.2.0 or higher
- **Dependency:** [Beyond Dimensions](https://www.curseforge.com/minecraft/mc-mods/beyond-dimensions)

## Development
This project is built using Forge Gradle 6.0 and utilizes the Beyond Dimensions API to interact with the storage network.
